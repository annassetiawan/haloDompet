# Mobile App Screenshots

Place your mobile app screenshots here to showcase on the landing page.

## Recommended screenshots:

1. **dashboard.png** - Main dashboard view showing wallets and total balance
2. **voice-recording.png** - Voice recording interface in action
3. **transaction-result.png** - Successful transaction result screen
4. **analytics.png** - Analytics/reports page
5. **advisor.png** - AI advisor chat interface

## Image specifications:

- Format: PNG or JPEG
- Dimensions: 1170x2532 (iPhone 14 Pro screen size) or similar mobile resolution
- File size: Keep under 500KB for optimal loading performance
- Use actual app screenshots for best results

## How to add screenshots:

1. Take screenshots from your device
2. Rename them according to the naming convention above
3. Place them in this directory
4. The landing page will automatically display them
